红宝网络在线打字系统的简介和使用说明：

本系统包含五大功能：中文打字练习、英文打字练习、中文打字比赛、英文打字比赛和强大的后台管理。本打字系统适用于各级各类学校、公司、企事业单位等进行打字比赛、打字练习。采用了ASP+ACCESS+VB+dll+JavaScript等技术开发，是基于B/S架构的，只需将本系统放在服务器上，其它电脑通过IE浏览器就可以在线打字练习或打字比赛了，使用非常方便。本系统能记录每次的打字成绩，打字比赛结束能马上汇总出比赛成绩和比赛名次。

一、最大亮点：
1、即指即译功能：在练习打字时，有鼠标即指即译功能：鼠标指向汉字时，就会显示该字的拼音和五笔编码（该功能适合正在学习中文输入的人群）；鼠标指向字母时，就会显示该字母的大写或小写（该功能适合一二年级学生学习大小写字母的输入）。
2、强大的分类汇总排名功能：系统会记录每个人每次的打字练习成绩，然后将每个人的打字成绩分类汇总，并排出名次。可以在本班（或本部门）排名，还可以在全校（或在单位）中排名。

二、安装说明：
1、将文件解压到IIS服务器上。
2、在服务器上双击regdll.bat进行组件注册。
3、输入正确的网址就可以使用了。

三、使用说明：
1、管理员用帐号admin密码admin888登录后：
①单击“用户帐号”，可以添加修改或删除用户的帐号密码（也提供了批量导入用户帐号密码的功能）。
②单击“文章分类”，可以添加修改或删除打字文章分类。
③单击“打字文章”，可以在某文章分类中修改或删除已有的打字文章内容，也可以在已有的分类中添加新的打字文章内容。
④单击“打字时间”，可以设定每次的打字练习比赛时间，在打字时若时间用完系统会自动提交成绩，然后退出打字。
⑤单击“开启比赛”，若要进行正式的打字比赛，请选择开启打字比赛，若不想比赛了，请选择关闭打字比赛。
⑥单击“所有成绩”，可以查阅每人每次的打字记录信息，也可以删除不要的记录。
⑦单击“成绩排名”，可以将每人的打字成绩分类汇总，算出每人的打字平均成绩，并排出名次，并能将成绩表导出成EXCEL表格。
⑧单击“软件注册”，按页面上的提示可以将本系统注册成正式版。
⑨单击“修改我的密码”，可以修改管理员的登录密码。

2、打字用户用自己的帐号和密码登录后：
①单击“中文练习”或“英文练习”，可以选择某分类下的某篇文章进行在线打字练习。
②单击“中文比赛”或“英文比赛”，可以选择某篇比赛文章进行在线打字比赛。在打字时，若时间用完系统会自动提交成绩，并结束打字。
③单击“我的成绩单”，可以查看自己每次的打字成绩记录。
④单击“修改我的密码”，可以修改自己的登录密码。
⑤单击页面左下角的“更多”，可以查看全校或全单位的打字成绩及排名。
⑥打字用户中若身份为“小管理员”，则可以帮本班（或本部门）忘记密码的用户找出密码。
